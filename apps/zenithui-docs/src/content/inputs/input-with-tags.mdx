# Input With Tags

An input component that allows users to type entries and hit Enter to convert them into removable tags, featuring smooth animations for adding and removing tags.

## 🚀 Installation through Shadcn CLI

<NPMTab path={`${process.env.NEXT_PUBLIC_SITE_URL}/r/input-with-tags.json`} />

Note: Make sure your Shadcn CLI is initialized in your project.

## Component Props List

| Prop          | Type     | Required | Description                                                                                      |
| :------------ | :------- | :------: | :----------------------------------------------------------------------------------------------- |
| `placeholder` | `string` |    No    | Placeholder text for the input field. Defaults to "Type something and press Enter...".           |
| `className`   | `string` |    No    | Optional additional CSS classes to apply to the container.                                       |
| `limit`       | `number` |    No    | Maximum number of tags allowed. Defaults to 10. Internally disables input when limit is reached. |
