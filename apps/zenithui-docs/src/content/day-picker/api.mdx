## Disabled Options

The `disabled` prop provides flexible options for disabling dates in the date picker. You can use it to restrict date selection based on various criteria, including specific dates, date ranges, days of the week, or custom logic.

```tsx
<DatePicker
  disable={{
    before: new Date(), // Disable past dates
    days: ["sunday"], // Disable Sundays
    dates: ["2025-04-01"], // Disable April Fools' Day
    modifier: (date) => {
      return date.getDate() % 2 === 0; // Disable every other day
    },
  }}
/>
```
