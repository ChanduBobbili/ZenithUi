# Customization

You can customize the toast in the way, either you can give `classNames` to the `ToasterProvider` or in `toastOptions` .

## class names to the ToasterProvider

```jsx
createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <ToastProvider
      theme="auto"
      classNames={{
        title: "text-white",
        description: "text-white",
        icon: "text-white size-5",
        actionButton: "bg-white text-black hover:bg-gray-200",
        closeButton: "bg-white text-black hover:bg-gray-200",
        cancelButton: "bg-white text-black hover:bg-gray-200",
      }}
    >
      <App />
    </ToastProvider>
  </StrictMode>,
)
```

---

## class names in the toast options

```jsx
<button
  onClick={() =>
    toast.warning("Warning Toast", {
      position: "top-left",
      showCloseButton: true,
      description: "Warning Description",
      classNames: {
        title: "text-white",
        description: "text-white",
        icon: "text-white size-5",
        actionButton: "bg-white text-black hover:bg-gray-200",
        closeButton: "bg-white text-black hover:bg-gray-200",
        cancelButton: "bg-white text-black hover:bg-gray-200",
      },
    })
  }
>
  Show Toast
</button>
```
