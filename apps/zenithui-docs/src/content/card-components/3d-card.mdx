# 3D Card Effect

The **3D Card Effect** is a visually striking UI component that creates a realistic 3D perspective effect when a user hovers over it. It tracks the mouse movement and smoothly rotates the card along the X and Y axes, elevating its inner elements to create a sense of depth and interactivity.

**Primary Use Cases:**

- Showcase feature highlights on landing pages.
- Highly interactive product or pricing cards.
- Portfolio items that demand attention.

_Context:_ This component is inspired by Aceternity UI and integrates seamlessly into the ZenithUI ecosystem.

## Component Props List

### `CardContainer`

| Prop Name            | Type              | Required | Default     | Description                                                                                   |
| :------------------- | :---------------- | :------- | :---------- | :-------------------------------------------------------------------------------------------- |
| `children`           | `React.ReactNode` | Yes      | -           | The content to be rendered inside the perspective container.                                  |
| `className`          | `string`          | No       | `undefined` | Additional Tailwind CSS classes applied to the inner animating wrapper.                       |
| `containerClassName` | `string`          | No       | `undefined` | Additional Tailwind CSS classes applied to the outermost wrapper controlling the perspective. |

### `CardBody`

| Prop Name   | Type              | Required | Default     | Description                                                          |
| :---------- | :---------------- | :------- | :---------- | :------------------------------------------------------------------- |
| `children`  | `React.ReactNode` | Yes      | -           | The components (usually `CardItem`s) to render inside the card body. |
| `className` | `string`          | No       | `undefined` | Tailwind CSS classes applied to the 3D preserved card wrapper.       |

### `CardItem`

| Prop Name    | Type                | Required | Default     | Description                                                                                      |
| :----------- | :------------------ | :------- | :---------- | :----------------------------------------------------------------------------------------------- |
| `children`   | `React.ReactNode`   | Yes      | -           | The content inside the elevated item.                                                            |
| `as`         | `React.ElementType` | No       | `"div"`     | The HTML tag or React component used to render the item (e.g., `"button"`, `"a"`, `"p"`).        |
| `translateX` | `number \| string`  | No       | `0`         | The distance to translate along the X-axis upon hover.                                           |
| `translateY` | `number \| string`  | No       | `0`         | The distance to translate along the Y-axis upon hover.                                           |
| `translateZ` | `number \| string`  | No       | `0`         | The distance to translate along the Z-axis upon hover. This dictates how "high" the item floats. |
| `rotateX`    | `number \| string`  | No       | `0`         | Angle to rotate around the X-axis upon hover.                                                    |
| `rotateY`    | `number \| string`  | No       | `0`         | Angle to rotate around the Y-axis upon hover.                                                    |
| `rotateZ`    | `number \| string`  | No       | `0`         | Angle to rotate around the Z-axis upon hover.                                                    |
| `className`  | `string`            | No       | `undefined` | Additional styles for the item.                                                                  |
