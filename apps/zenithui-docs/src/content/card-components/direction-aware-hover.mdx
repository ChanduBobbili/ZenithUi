# Direction Aware Hover

The **Direction Aware Hover** component intelligently detects the direction from which the user's mouse enters the bounding box of the card (top, right, bottom, or left). It then animates the image scaling and text overlay to slide in from that exact direction.

**Primary Use Cases:**

- Interactive image galleries.
- Product feature displays where hover transitions need a touch of spatial cleverness.
- Blog post thumbnails revealing article titles smoothly.

_Context:_ Built using mathematical detection via cursor coordinates mapped to the element's bounding rect, passing the result to Framer Motion states.

## Component Props List

### `DirectionAwareHover`

| Prop Name           | Type                        | Required | Default     | Description                                                                   |
| :------------------ | :-------------------------- | :------- | :---------- | :---------------------------------------------------------------------------- |
| `imageUrl`          | `string`                    | Yes      | -           | Absolute or relative URL of the background image to render.                   |
| `children`          | `React.ReactNode \| string` | Yes      | -           | The overlay content (e.g., text, buttons) that is revealed upon hover.        |
| `childrenClassName` | `string`                    | No       | `undefined` | Custom Tailwind CSS classes applied to the wrapper containing the `children`. |
| `imageClassName`    | `string`                    | No       | `undefined` | Custom Tailwind CSS classes applied directly to the `<img />` tag.            |
| `className`         | `string`                    | No       | `undefined` | Additional styles applied to the outermost container of the component.        |
