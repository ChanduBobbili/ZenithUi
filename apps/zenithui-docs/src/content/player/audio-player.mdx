# Audio Player

A beautifully designed audio player component built with Framer Motion that supports cover art, shuffle, repeat controls, and a fully interactive progress slider.

## 🚀 Installation through Shadcn CLI

<NPMTab path={`${process.env.NEXT_PUBLIC_SITE_URL}/r/audio-player.json`} />

Note: Make sure your Shadcn CLI is initialized in your project.

## Component Props List

| Prop    | Type     | Required | Description                                        |
| :------ | :------- | :------: | :------------------------------------------------- |
| `src`   | `string` |   Yes    | The URL or relative path to the audio source file. |
| `title` | `string` |    No    | Optional title of the audio file to display.       |
| `cover` | `string` |    No    | Optional cover image URL to display.               |
