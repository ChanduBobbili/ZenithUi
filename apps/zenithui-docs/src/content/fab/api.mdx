## Menu components

Use these inside **FabContent**.

### FabGroup

Wraps a set of items. Accepts `className` and standard `div` props.

### FabLabel

Section label. Accepts `className` and standard `div` props. Use for accessibility and section titles.

### FabSeparator

Visual divider. Accepts `className` and standard `div` props.

### FabItemIndicator

Renders when the parent **FabCheckboxItem** or **FabRadioItem** is checked/selected. Accepts `className` and `children` (e.g. a check icon). Style with `group-data-[state=checked]:inline-flex` when the parent has the `group` class.
