## Positions

Set `position` on `Fab.Root` to place the trigger. Use the dropdown to try corners, edges, and center. Use `xOffset` and `yOffset` to nudge from the edge.

<FabDemoWrapper
  title="Positions"
  description="Change the dropdown to move the FAB to different viewport positions."
>
  <PositionsDemoFab />
</FabDemoWrapper>

**Positions:** `top-left`, `top-center`, `top-right`, `center-left`, `center`, `center-right`, `bottom-left`, `bottom-center`, `bottom-right`.

---

## Content placement

Set `placement` on `Fab.Content` to open the panel **relative to the trigger**: `top`, `bottom`, `left`, or `right`. The component flips automatically if there isnâ€™t enough space.

In the [main FAB page](/fab/fab), the content uses `placement="top"` so the menu opens above the trigger. For a FAB at `bottom-right`, that keeps the menu in view; for `top-right`, you might use `placement="bottom"`.
