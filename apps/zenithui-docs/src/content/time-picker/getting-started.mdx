# Getting Started

The ZenithUI Time Picker is a React component that provides a smooth and accessible interface for selecting time. It features a scrollable design for hours, minutes, and AM/PM selection. The component supports both 12-hour and 24-hour formats and offers seamless customization options for a modern user experience.
