# Count Down

The `CountDownTimer` component provides a customizable countdown timer with options for labels, descriptions, and an expiry callback.

## üé® Custom Styling

```tsx
import { CountDownTimer } from "@zenithui/time-picker";

<CountDownTimer
  startTime={new Date().toISOString()}
  className="font-bold text-blue-500"
/>
```

<Callout type="info">
  Import your styles.css/global.css after the `app.jsx`, Hence your styles will
  have higher specificity
</Callout>

```jsx {2}
import CountDownTimer from "@zenithui/time-picker"
import "./styles.css"
```
